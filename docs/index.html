Create a new API key

​

Creates a new API key for authentication. Can create a new tenant atomically or use an existing tenant.

Body
required
application/json
  • expiresAt
    Type: string | nullFormat: date-time
  • name
    Type: string | null
  • projectId
    Type: string | nullFormat: uuid
  • tenantId
    Type: string | nullFormat: uuid
  • tenantName
    Type: string | null
Responses
  • 200

    OK

Request Example for post/cerberus/api-keys
curl https://instructo.ro/cerberus/api-keys \
  --request POST \
  --header 'Content-Type: application/json' \
  --header 'Authorization: Bearer YOUR_SECRET_TOKEN' \
  --data '{
  "name": null,
  "tenantId": null,
  "tenantName": null,
  "projectId": null,
  "expiresAt": null
}'
No Body

OK
